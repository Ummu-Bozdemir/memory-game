# Memory Game

A multi-level memory matching game built with **Phaser.js**.

The goal of the game is to find all matching card pairs with the fewest moves and in the shortest time.

---

##  How to Play

1. Click on a card to flip it.
2. Flip a second card to find its matching pair.
3. If the cards match, they stay open.
4. If they do not match, they flip back after a short delay.
5. Match all pairs to complete the level.
6. Each new level increases the number of cards.

You can restart the current level at any time using the **Restart** button.

---

##  Features

- Multi-level gameplay
- Increasing difficulty per level
- Timer to track elapsed time
- Move counter
- Restart button
- Dynamic grid scaling
- Clean and minimal UI

---

##  Technologies Used

- JavaScript (ES6)
- Phaser.js (game framework)
- HTML5 & CSS
